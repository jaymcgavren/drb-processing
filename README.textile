h1. DRb-Processing

Ruby-Processing, Internet-enabled.


h1. Installation

<pre>
  gem install drb-processing
</pre>


h1. Running a Server

<pre>
  drb-processing_server
</pre>


h1. Connecting to a Server

drb-processing &lt;server_url&gt;
