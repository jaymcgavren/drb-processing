h1. DRb-Processing

Ruby-Processing, Internet-enabled.


h1. Installation

<pre>
  gem install ruby-processing drb-processing
</pre>


h1. Running a Server

<pre>
  drbpd
</pre>


h1. Connecting to a Server

<pre>
  rp5 live bin/drbp druby://&lt;server_address&gt;:9000
</pre>
